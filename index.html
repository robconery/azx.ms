Build Your Azure Scripts Easily

Choose what you want, copy, paste. Don't see what you want? Let us know. Our PRs are open as well.

Install AZX in bash: 

echo "function azx() { curl -sL https://azx.ms/api/\$1;}" >> \
~/.bashrc && source ~/.bashrc

Install AZX in zshell: 

echo "function azx() { curl -sL https://azx.ms/api/\$1;}" >> \
~/.zshrc && source ~/.zshrc

Run it: 

azx help
azx list
azx postgres > azure_postgres.sh
